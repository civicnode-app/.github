## 📘 Project Overview

<p align="justify">
CivicNode AI adalah platform pengawasan lingkungan cerdas berbasis Computer Vision dan Blockchain yang dirancang untuk mengatasi krisis pengelolaan limbah di area publik. Mengusung misi Planetary Health, platform ini mentransformasi CCTV konvensional menjadi instrumen penegak akuntabilitas publik yang bekerja secara otomatis dan transparan.
</p>
---

## ⭐ Key Features

- ​**Universal Hardware Integration**: Mampu menerima input video streaming dari berbagai perangkat melalui protokol R
- **Smart Snapshot Engine:** Melakukan pengambilan gambar (snapshot) secara otomatis untuk diproses oleh model AI guna efisiensi komputasi.
- **Hybrid Authentication Portal:** Menyediakan pintu masuk untuk umum dan administrator.
- **Dashboard Analytic:** Bukti pelanggaran yang terdeteksi wajib diunggah ke IPFS dan hash-nya dicatat secara permanen.
- **Frontend Portal:** Dibangun dengan React/Next.js untuk pengalaman pengguna maksimal.
---

## 🏆 Proyek Utama: Predictive Lead Scoring Portal
Platform ini dirancang untuk memberikan:

- **Dashboard interaktif** berbasis React/Next.js
- **API terintegrasi** (FastAPI)
- **Manajemen data** menggunakan PostgreSQL/MySQL
